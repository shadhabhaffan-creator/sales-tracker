# AuraSales | Premium Business Management Suite

A world-class private Sales & Accounting Tracker with cinematic glassmorphic visuals and role-based access control.

## 📁 Project Structure

- **`frontend/`**: Next.js 15 app with React, TypeScript, Tailwind CSS, and Framer Motion.
- **`backend/`**: Node.js & Express REST API with JWT authentication and role middleware.
- **`database/`**: Mongoose models and schema definitions shared across the monorepo.
- **`docs/`**: API documentation and architectural guides.

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 18+
- MongoDB (Local or Atlas)

### 2. Backend Setup
```bash
cd backend
npm install
npm run dev
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
